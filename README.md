## Slider Component Theme Setup

1. **Install Advanced Custom Fields (ACF)**  
   - Log in to the WordPress dashboard.  
   - Navigate to `Plugins → Add New`, search for “Advanced Custom Fields”.  
   - Click `Install Now`, then `Activate`.

2. **Activate the Slider Component theme**  
   - Go to `Appearance → Themes`.  
   - Locate “Slider Component” in the list (upload it first if necessary).  
   - Click `Activate`.

3. **Review the sample page**  
   - After activation the theme automatically creates a published page called “Slider Sample”.  
   - Visit `/slider-sample/` on the site front-end to confirm the slider renders.

4. **Customize the slider content**  
   - Edit the “Slider Sample” page (or any other page using the “Slider Page” template).  
   - Update the ACF fields: title, background color, WYSIWYG text, up to three images, and their captions.  
   - Publish or update the page to see the changes reflected on the front-end.
